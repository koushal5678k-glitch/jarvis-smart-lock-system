# Jarvis Smart Lock System

An AI-powered authentication system that combines Voice Recognition and Face Recognition.

## Features

- Voice Activated Commands
- Face Detection
- Face Recognition
- Smart Greetings
- Unknown Person Alert
- Multi-Factor Authentication

## Technologies Used

- Python
- OpenCV
- face_recognition
- SpeechRecognition
- pyttsx3

## Commands

- Jarvis
- Who am I
- Open Camera
- Start Detection

## Future Scope

- Liveness Detection
- Mobile Deployment
- Smart Home Integration
